[![Coveralls Status][coveralls-image]][coveralls-url]

> pure js decrement for big numbers

## Install

    npm install --save bignum-dec

## Usage

```js
import dec from 'bignum-dec';

dec('420000000000000000000000'); // '419999999999999999999999'
```
